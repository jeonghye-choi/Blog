- **Problem**

  ```bash
  TypeError: Cannot read properties of null (reading 'useContext')
    at exports.useContext 
  ```

- **Solved**


  ```ts
  const alertDispatch = useSnackBarsDispatch();

  export default function useContact() {
    // ...
  }
  ```
  `useSnackBarsDispatch()`를 바깥쪽에 정의했기 때문에 발생한 문제였다.
  
  `useContact()` function 안에 `useSnackBarsDispatch()`, 즉 `useContext()`를 정의해야 한다.
