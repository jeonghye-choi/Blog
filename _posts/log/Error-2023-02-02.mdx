- **Problem**

  emailjs를 사용 중, 환경변수로 사용한 public key가 유효하지 않다는 400 에러 메세지가 떴다.

  ```bash
  POST https://api.emailjs.com/api/v1.0/email/send-form 400

  Error: The Public Key is invalid. To find this ID, visit https://dashboard.emailjs.com/admin/account'
  ```
  public key가 유요하지 않다니.. 이게 무슨 말인가...
  몇 시간 동안 이유를 찾아 문서를 뒤지고 구글링도 해봤는데 결국 생각지도 못했던 문제 때문이었다. 

- **Solved**

환경 변수 설정에 문제가 있었다.

`.env`파일에 환경변수를 설정할 때, 사용하는 프레임워크(또는 라이브러리)에 따라
규칙이 존재한다.

- react : `REACT_APP_{변수}`와 같이 `REACT_APP`을 변수 앞에 붙여줘야 한다.
- nextjs : `NEXT_PUBLIC_{변수}`와 같이 `NEXT_PUBLIC`을 변수 앞에 붙여줘야 한다.
