## 자바스크립트의 표준

JavaScript는 모질라 코퍼레이션의 공동 설립자이자 CTO를 맡았던, Brendan Eich에 의해 만들어졌습니다.

자바스크립트의 초기 버전은 Brendan Eich가 초기 브라우저인 네스케이프에 탑재하기 위해서 만들어졌고, 
인터넷 익스플로러도 JScript라고 이름 붙인 자바스크립트를 지원하기 시작했습니다. 
두 언어가 분화되어 크로스 브라우징 이슈를 만드는 것을 막기 위해 네스케이프는 자바스크립트 표준화를 ECMA 국제화 기구에 제안했습니다.

그렇게 자바스크립트는 더 이상 Brendan Eich 혼자가 만드는 언어가 아닌 위원회가 만드는 언어가 되었습니다.

## ECMAScript

- ECMA 국제화 기구에 의해, ECMAScript라는 새로운 언어 표준이 생겼습니다. 

- ES는 ECMAScript의 약어로, ES1, ES2와 같이 버전을 함께 표현합니다.

- [다음 링크](https://www.w3schools.com/js/js_versions.asp)를 통해 각 버전바다 어떤 변화가 있었는지 확인할 수 있습니다.


## ES5 vs ES6

ES5와 ES6 사이에는 버전 차이에 따른 기능 추가가 뚜렷합니다. 따라서 이 둘의 구분이 필요합니다.
둘 사이에는 6년이라는 시간 차이가 있는데, 이 사이에 많은 기능들이 추가 되었습니다.

ES6+, 즉 ES6이후를 **모던 자바스크립트**라고 부릅니다.

- Template literal 템플릿 리터럴 추가
- 화살표 함수
- this
- 변수 선언
- 모듈
- 클래스
- let, const 키워드 추가
- default parameter 추가
- 구조분해 할당
- 프로미스
- string 메서드
